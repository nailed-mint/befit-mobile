# befit-mobile
BeFit, una aplicación móvil para registrar entrenamientos y progreso físico a lo largo del tiempo.

## ✨ Funcionalidades
- Registro de entrenamientos
- Visualización de progreso
- Autenticación de usuarios

## 🚀 Getting Started

```bash
npm run android
```
or
```bash
npm run ios
```

## 🛠️ Tech Stack
- React Native
- TypeScript
- Expo
- ESLint + React plugins
